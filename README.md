Project: SIMPLE KEYLOGGER
Objective:

The primary objective of this project is to understand the working concept of keystroke logging in a controlled and ethical environment. The project aims to simulate a basic input logging mechanism within an application interface to study how keylogging techniques function. It also focuses on creating awareness about cybersecurity threats and highlighting the importance of ethical responsibility while handling user data.
